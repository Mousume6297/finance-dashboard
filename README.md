# 💰 Finance Dashboard UI

## 📌 Overview

This project is a **Finance Dashboard UI** built using React and Tailwind CSS.
It helps users track financial activity, view insights, and explore transactions through a clean and interactive interface.

This project focuses on frontend design, state management, and user experience.

## 🛠 Tech Stack

* React.js (Vite)
* JavaScript
* Tailwind CSS
* Recharts (for charts)

---

## ✨ Features

### 📊 Dashboard Overview

* Total Balance, Income, and Expenses cards
* Balance trend (Line Chart)
* Spending breakdown (Pie Chart)

### 📋 Transactions

* View all transactions in a table
* Search by category
* Filter by income/expense
* Handles empty state (No data)

### 👤 Role-Based UI

* Viewer → can only view data
* Admin → can add transactions (UI simulation)

### 💡 Insights Section

* Highest spending category
* Total income and expense summary

### 🎨 UI/UX Enhancements

* Clean and responsive design
* Works on mobile and desktop
* Smooth hover animations

---

## ⚡ Additional Features

* 🌙 Dark Mode toggle
* 💾 Local Storage (data persists after refresh)
* 📤 Export transactions as CSV
* 🎬 Smooth hover and transition effects

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/finance-dashboard.git
```

2. Navigate to project:

```bash
cd finance-dashboard
```

3. Install dependencies:

```bash
npm install
```

4. Run the project:

```bash
npm run dev
```
